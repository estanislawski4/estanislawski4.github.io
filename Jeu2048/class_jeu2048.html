Description détaillée

Représente le jeu 2048 avec ses fonctionnalités principales.

La classe Jeu2048 gère la logique du jeu 2048, y compris le placement de nouvelles tuiles, le déplacement des tuiles, le calcul du score, la vérification de l'état du jeu, etc.

◆ PlacerNouvelleTuile()

void Jeu2048::PlacerNouvelleTuile ( )

Place une nouvelle tuile aléatoire sur la grille.

Place une nouvelle tuile aléatoire sur la grille du jeu 2048.

Cette fonction génère aléatoirement des coordonnées (x, y) sur la grille du jeu et place une nouvelle tuile (2 ou 4) sur la case correspondante si elle est vide.

La génération des coordonnées et la valeur de la tuile sont déterminées de manière aléatoire.

Précondition
La fonction ToutesLesCasesOccupees() doit renvoyer false pour qu'une nouvelle tuile soit placée.
Postcondition
Une nouvelle tuile est placée sur la grille du jeu à une position aléatoire.
