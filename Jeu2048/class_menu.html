Description détaillée

Gère l'affichage et la sélection d'options dans un menu.

La classe Menu permet de créer et d'afficher des menus avec différentes options, ainsi que de récupérer le choix de l'utilisateur.

◆ AfficherEtAttendreChoix()

int Menu::AfficherEtAttendreChoix ( )

Affiche le menu, attend le choix de l'utilisateur et le retourne.

Affiche le menu et attend le choix de l'utilisateur.

Renvoie
Le choix de l'utilisateur.

Cette méthode affiche le menu avec les options numérotées et attend que l'utilisateur fasse un choix valide. L'utilisateur doit entrer un nombre entre 1 et le nombre total d'options inclus.

Renvoie
Le choix de l'utilisateur, ou -1 si une entrée invalide a été détectée.
Précondition
Le tableau d'options ne doit pas être vide et doit être non nul. Le nombre d'options doit être différent de zéro.
Postcondition
Le menu est affiché dans la console et le choix de l'utilisateur est retourné. Si une entrée invalide est détectée, -1 est retourné.

Attend l'appui d'une touche pour continuer.

Postcondition
