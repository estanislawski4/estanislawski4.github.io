◆ onSendButtonClicked

void Interface::onSendButtonClicked ( )
privateslot

Gère l'envoi d'une trame.

Construit et envoie une trame APRS et une trame LoRa, effectue leur conversion et leur stockage dans la base de données. Les messages de statut sont affichés dans le log.

Vérifie que le message à envoyer n'est pas vide, construit et envoie une trame APRS et une trame LoRa. La trame LoRa est convertie en trame AX.25 avant d'être transmise via le port série, puis stockée en base.
