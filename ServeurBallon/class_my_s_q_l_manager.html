Description détaillée

Gestionnaire de connexion et d'opérations MySQL.

La classe MySQLManager permet d'ouvrir et de fermer la connexion à une base de données MySQL, d'exécuter des requêtes SQL (lecture et modification) et de réaliser des opérations spécifiques liées à la gestion des machines dans la base de données.
