Description détaillée

Gestionnaire de trames KISS.

La classe KISSHandler permet de traiter les données brutes reçues au format KISS. Elle décode ces données pour extraire une trame AX.25, la convertit en format TNC2, et transmet des informations via des signaux, notamment pour l'envoi de trames vers APRS-IS et pour la notification de réception de trames LoRa.

◆ parseKISSData()

void KISSHandler::parseKISSData ( const QByteArray &  data)

Analyse et décode des données reçues au format KISS.

Traite le flux de données KISS en découpant les trames, en gérant les séquences d'échappement et en transmettant chaque trame complète pour traitement.

Paramètres
dataLes données brutes au format KISS.

Parcourt le flux de données, gère les délimiteurs de trame (0xC0), les séquences d'échappement (0xDB, 0xDC, 0xDD), et reconstitue les trames complètes. Lorsqu'une trame complète est reconstituée, elle est transmise à la fonction processKISSFrame.

Paramètres
dataLes données brutes au format KISS.

◆ processKISSFrame()

void KISSHandler::processKISSFrame ( const QByteArray &  frame)
private

Traite une trame KISS complète.

Décode la trame KISS en extrayant le port et le payload, convertit la trame AX.25 en format TNC2, puis émet des signaux pour la journalisation et, si activé, pour l'envoi vers APRS-IS.

Paramètres
frameLa trame KISS complète à traiter.

Extrait le byte de port et le payload AX.25 de la trame, convertit ce dernier en format TNC2 via l'objet AX25Converter, et émet des signaux pour la journalisation. Si la conversion est réussie, la trame TNC2 est également émise via le signal loRaFrameReceived et, si activé, envoyée vers APRS-IS.

Paramètres
frameLa trame KISS complète à traiter.
